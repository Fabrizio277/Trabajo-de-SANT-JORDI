# Rosa de Sant Jordi · Versió estàtica

Projecte web senzill per a la pràctica de Sant Jordi.

## Tecnologies
- HTML
- CSS
- JavaScript
- localStorage

## Funcionalitat
- Landing page responsive
- Formulari per crear una rosa virtual
- Frase aleatòria segons el to seleccionat
- URL compartible amb la targeta generada
- Galeria amb exemples i últimes roses creades en el navegador

## Execució
Només cal obrir `index.html` en un navegador.

## Nota tècnica
No cal backend ni base de dades. La informació de la targeta es passa per URL i la galeria local es desa a `localStorage`.
